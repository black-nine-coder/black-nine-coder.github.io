## vs code를 git diff, merge tool 로 설정하기
1. git config --global core.editor "code --wait"
1. git config --global -e
1. ~/.gitcofig
1. diff tool 설정
```bash
    [diff]
        tool = vscode
    [difftool "vscode"]
        cmd = code --wait --diff $LOCAL $REMOTE
```

1. Display에 표시한다.
```bash
    [merge]
        tool = vscode
    [mergetool "vscode"]
        cmd = code --wait $MERGED
```
